## Sistema de Evolução dos Personagens

| Nível | Nome       | Desbloqueios                               |
|-------|------------|--------------------------------------------|
| 1     | Aprendiz   | Animação base                              |
| 2     | Explorador | Nova animação de vitória                   |
| 3     | Mestre     | Roupa/acessório especial + novas falas     |
| 4     | Guardião   | Efeitos visuais exclusivos                 |
| 5     | Lendário   | Animação lendária + fala épica             |

Evolução baseada no XP acumulado pelo aluno em cada mundo vinculado ao personagem.
